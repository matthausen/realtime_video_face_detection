# Real time video face detection

This program uses the haarcascade pretrainel xml model provided by OpenCV


### Build &  run:

- `cmake .`

- `make`

- `./main`


### Dependencies:

Cmake and OpenCV

- `brew install cmake`

- `brew install opencv`

Pre-trained models available here:

https://github.com/opencv/opencv/tree/3.4/data
